# Argos – Akıllı Güvenlik ve Erişim İzleme Platformu

Web tabanlı güvenlik ve erişim izleme platformu tanıtım sayfası.

## Özellikler

- Gerçek zamanlı erişim takibi ve güvenlik analizi
- Erişim loglama, raporlama, öneriler
- Soft gri arka plan, yıldız animasyonu, responsive tasarım

## Çalıştırma

`index.html` dosyasını tarayıcıda açın veya yerel sunucu ile:

```bash
npx serve
```

## Yapı

- `index.html` – Ana sayfa
- `styles.css` – Stiller
- `assets/` – Logo ve görseller
